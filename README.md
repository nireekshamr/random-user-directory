# Random User Directory

## 📌 Project Overview

The Random User Directory is a responsive web application built using HTML, CSS, and JavaScript. It fetches random user data from the Random User API and displays user profiles in an interactive and user-friendly interface.

---

## 🚀 Features

- Fetch random user data from the Random User API
- Display user profile cards
- Search users by name
- Reload button to fetch a new set of users
- Responsive design for desktop and mobile
- Clean and modern interface

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Random User API

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open **index.html** in your web browser.
4. Ensure you have an internet connection to fetch user data from the API.
